# 🚀 Hackathon Resume Builder

My **first-ever hackathon project** — a dynamic and interactive **Resume Builder** built with **TypeScript**, allowing users to create professional resumes, preview them live, download them as PDF, and share them through a public link.

This project was a major milestone in my developer journey because it was built during my **first hackathon**, where I learned how to build and deploy a complete product under real deadline pressure.

# 📌 Project Overview

The Resume Builder helps users quickly create professional resumes by filling out a dynamic form.

As users type their information, the resume updates **in real time**, giving instant feedback and a smooth user experience.

Once complete, users can:

- Download their resume as a **PDF**
- Share it using a **public link**
- Access it directly from the browser
